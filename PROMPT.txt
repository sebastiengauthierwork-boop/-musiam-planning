Ajouter un bouton "Exporter la base" dans /admin pour l'administrateur :

1. PAGE /admin/utilisateurs ou /parametrage, ajouter un bouton "Sauvegarder les donnees" visible uniquement par l'admin

2. Quand on clique, exporter TOUTES les tables en un seul fichier JSON telechareable :
- sites, teams, employees, employee_teams, shift_codes, absence_codes, schedules, cycle_schedules, users, staffing_structures, staffing_structure_positions, annual_calendar, planning_archives, planning_status, job_functions, role_permissions, contacts_utiles, employee_history

3. Le fichier JSON est telecharge automatiquement avec le nom : backup_planekipe_YYYY-MM-DD.json

4. Ce n'est pas un import automatique, c'est juste une sauvegarde de secours. En cas de probleme on pourra reconstruire les donnees a partir de ce fichier.
