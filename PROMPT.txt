Vue mobile admin - consultation des plannings de toutes les equipes :

1. PAGE /mon-planning POUR LES ADMINS :
Quand un admin accede a /mon-planning (via le choix smartphone ou via la sidebar) :
- En haut, afficher deux dropdowns :
  a) "Site" : liste de tous les sites (l'admin voit tout)
  b) "Equipe" : liste des equipes du site selectionne
- Quand un site et une equipe sont selectionnes, afficher le planning de cette equipe (meme vue que l'onglet "Mon equipe" des salaries)
- L'admin peut naviguer entre les mois (mois en cours et mois suivant)
- Consultation uniquement, pas de modification depuis le smartphone

2. PLANNING PERSONNEL DE L'ADMIN :
- Si l'admin a un employee_id dans la table users (il est aussi salarie sur un site), afficher un onglet supplementaire "Mon planning" avec son planning personnel
- Si l'admin n'a PAS de employee_id, ne pas afficher l'onglet "Mon planning" personnel, uniquement la consultation des equipes

3. POUR LES RESPONSABLES :
Meme logique mais le dropdown Site est limite a son site. Le dropdown equipe montre toutes les equipes de son site.

4. POUR LES MANAGERS :
Pas de dropdown site. Le dropdown equipe montre uniquement ses equipes autorisees.

5. POUR LES SALARIES :
Pas de changement. Pas de dropdown. Ils voient leur planning et celui de leur equipe.
