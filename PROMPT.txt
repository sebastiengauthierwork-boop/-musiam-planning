Ajouter un champ "Date d'entree" sur les employes :

1. Ajouter la colonne dans Supabase : genere le SQL et copie avec clip :
ALTER TABLE employees ADD COLUMN IF NOT EXISTS start_date DATE;

2. Dans le formulaire de creation/modification d'employe sur /employes, ajouter un champ date "Date d'entree" (date picker). Ce champ est optionnel.

3. IMPACT SUR LE PLANNING : quand on affiche la grille de saisie d'un mois, si un employe a une date d'entree APRES le 1er du mois :
   - Les jours AVANT sa date d'entree sont grisees et non editables
   - Il n'apparait PAS dans le planning des mois precedant sa date d'entree
   - Il apparait dans le planning du mois de son entree mais avec les jours avant sa date d'entree bloques

4. Meme logique pour le planning imprimable, le compteur d'heures et la feuille d'emargement : ne pas compter les jours avant la date d'entree

5. Ajouter aussi un champ "Date de sortie" (optionnel) avec la meme logique inverse : apres la date de sortie, les jours sont grisees et non editables
