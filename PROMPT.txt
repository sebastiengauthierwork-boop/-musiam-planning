4. COULEURS PAR CODE HORAIRE - PAR STATUT ET HORAIRE :
Les couleurs sont determinees par la nomenclature du code (1ere lettre = statut, derniere lettre = type horaire) :

MANAGERS (codes commencant par M) :
- Ouverture (finit par O) : Bleu moyen (#4A90C4), texte blanc
- Milieu (finit par M) : Bleu clair (#6BAED6), texte noir
- Fermeture (finit par F ou S) : Bleu fonce (#2C5F8A), texte blanc

EMPLOYES (codes commencant par E) :
- Ouverture (finit par O) : Vert moyen (#5BA55B), texte blanc
- Milieu (finit par M) : Vert clair (#7BC47B), texte noir
- Fermeture (finit par F ou S) : Vert fonce (#3A7A3A), texte blanc
- Livraison (contient L) : Orange brique (#C47A4A), texte blanc

CADRES (codes commencant par C ou codes P, P/O, P/F) :
- P : Gris bleu (#8899AA), texte blanc
- P/O : Gris bleu clair (#A0B0C0), texte noir
- P/F : Gris bleu fonce (#667788), texte blanc

ABSENCES ET REPOS :
- R (Repos) : fond NOIR (#000000), texte blanc
- CP, JF, RTT, RSD : fond GRIS moyen (#888888), texte blanc
- M (Maladie), AT, CSS et toutes les autres absences : fond ROUGE BRIQUE (#A83232), texte blanc
