Distinction superadmin et admin :

1. SUPERADMIN (toi uniquement, sgauthier@musiam-paris.com) :
- Acces total a TOUT sans exception
- Seul a pouvoir creer, modifier, supprimer des utilisateurs dans /admin/utilisateurs
- Seul a pouvoir modifier les permissions (Roles et acces)
- Seul a pouvoir desarchiver les plannings

2. ADMIN (ex: controleuse de gestion) :
- Acces complet a tous les sites et equipes
- Peut voir et modifier les plannings, salaries, equipes, codes horaires, cycles, parametrage
- Peut archiver les plannings
- Peut exporter les donnees
- Peut imprimer tout
- NE peut PAS acceder a /admin/utilisateurs (la page est masquee dans la sidebar pour les admins)
- NE peut PAS creer, modifier ou supprimer des utilisateurs
- NE peut PAS modifier les permissions (Roles et acces masque)
- NE peut PAS desarchiver les plannings

3. DANS LE CODE :
- Partout ou il y a role === 'admin', ajouter role === 'superadmin' pour que le superadmin garde tous les acces
- La page /admin/utilisateurs : verifier role === 'superadmin' au lieu de role === 'admin'
- La sidebar masque "Utilisateurs" si role !== 'superadmin'
- Le bouton "Creer acces" sur la page Salaries : visible uniquement pour le superadmin
