Acces au planning personnel pour les admins, responsables et managers :

1. APRES LE LOGIN pour admin, responsable et manager :
- Si l'ecran est un smartphone (largeur < 768px) : afficher un ecran de choix avec 2 boutons :
  a) "Mon planning" : redirige vers /mon-planning (meme vue que les salaries, avec son planning perso et le planning equipe)
  b) "Gestion" : redirige vers /tableau-de-bord (l'interface complete habituelle)
- Si l'ecran est un PC/tablette (largeur >= 768px) : rediriger directement vers /tableau-de-bord comme d'habitude

2. LIEN "MON PLANNING" DANS LA SIDEBAR :
- Ajouter un lien "Mon planning" en haut de la sidebar pour les admins, responsables et managers
- Ce lien ouvre /mon-planning avec leur planning personnel (comme un salarie)
- Pour que ca fonctionne, il faut que le user admin/responsable/manager ait un employee_id dans la table users (lie a leur fiche salarie)

3. SUR /mon-planning :
- Si le user est admin/responsable/manager, ajouter un bouton "Retour a la gestion" en haut qui redirige vers /tableau-de-bord
- Les salaries ne voient PAS ce bouton

4. ASSOCIATION EMPLOYEE_ID :
- Dans le formulaire de creation/modification d'utilisateur (/admin/utilisateurs), quand le role est admin, responsable ou manager, ajouter un dropdown optionnel "Salarie associe" (comme pour les salaries)
- Ca permet de lier le compte utilisateur a une fiche salarie pour voir son planning perso
